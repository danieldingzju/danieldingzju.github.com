JavaBeans(TM)

JavaBeans是为了定义出最小的可复用的组件，具备移植性和复用性，类似com+组件。

Java Bean由properties，event，method(listen methods)组成。

JavaBeans定义了一堆规范和利用了listener design pattern来完成properties变化事件和自定义事件的监听。
